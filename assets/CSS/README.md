### ✧ Campaign Identity ✧
The **Silakbo** aesthetic is a high-energy "Neon-Pink" design built for impact. 
It utilizes **Bebas Neue** for strength and **Montserrat** for clarity, 
creating a professional yet vibrant digital presence.

### 📂 Repository Structure
> *The architectural backbone of the Silakbo campaign site.*

* **`index.html`** — Core structure including the Hero video and Candidate grid.
* **`style.css`** — The complete design system, neon animations, and mobile repairs.
* **`Departmentsl css's`** — The complete design system for each dept, neon animations, and mobile repairs.
* **`/assets`** — High-resolution candidate portraits and university logos.
* **`README.md`** — This documentation and technical overview.
