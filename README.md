## 🌸 <span style="color:#e91e63;">Welcome</span>

Welcome to the **Silakbo Static Website Repository**. 

This space serves as a digital home for our **candidates, credentials, and platforms**. We built this to ensure that student voters have a single, accessible hub to explore our vision without getting lost in a sea of social media posters.

## ⏱ <span style="color:#2196f3;">Development Context</span>

This project was a **72-hour sprint**. Developed under significant time constraints, the focus was on high-impact delivery:
* ⚡ **Rapid Deployment:** From concept to live link in 3 days.
* 🎯 **Centralized Info:** Profiles, platforms, and credentials in one place.
* 🎨 **Visual Branding:** Maintaining the "Silakbo" aesthetic across all pages.
